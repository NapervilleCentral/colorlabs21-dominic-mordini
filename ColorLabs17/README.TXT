PROJECT TITLE: Poster Project (image name is "finalPoster")
PURPOSE OF PROJECT: To adjust and change the pixels and colors of an image to create a modified texture for each image
VERSION or DATE:5/14/2026
HOW TO START THIS PROJECT:

I used multiple different methods to manipulate myn original image. I used vertical mirror and horizontal mirror to give
the creature in my image multiple heads and limbs. I used a method that inverted the colors to make a coll effect with the image.
I also used s sepia filter to make my image look old and vintage. I also used a recursive method to create a picture inside a picture illusion. I used
all of these to create an interesting 6 picture collage.
